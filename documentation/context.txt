[ Glossary ]

/Metrics
	The measurements collected described the effects of the tests provided by a project.

/Harness
	In the context of &.libtest and &.libharness, a harness is an
	object used to manage execution of the tests. Harnesses define how and
	what tests are executed.

/Seal
	Refers to the execution of a test: seal the fate of a test.

/Factor
	A part of a &Product. An addressable part of a project.

/Project
	A set of factors making a &Product.

/Product
	The conceptual effect of a project. Often project and product can be used
	interchangeably, but product is usually referring to the conceptual collection
	of factors.
